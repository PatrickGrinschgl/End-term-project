# PPGIS Urban Experiences – Graz Innere Stadt

This project allows participants to map **pedestrian comfort and stress** in Graz Innere Stadt.

## How to use
1. Center the red crosshair on a location.
2. Select experience type, reason, and leave a comment.
3. Click "Save Location".
4. Download collected data as GeoJSON.

## File Structure
- `index.html` – main page
- `src/css/style.css` – styles
- `src/js/script.js` – JS logic
- `data/innere_stadt.geojson` – district boundary
